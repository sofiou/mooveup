# Moove Up

Site vitrine + maquette interactive pour le projet Moove Up — une solution sportive hybride pensée pour les Parisiens actifs.

---

## C'est quoi ?

Moove Up combine trois choses dans une seule app :
- Une **application mobile** qui génère un programme d'entraînement sur mesure
- La **location de kits sportifs** livrés à domicile ou récupérables en box QR
- L'accès à des **salles de sport intimistes** à Paris (max 15 personnes)

Ce repo contient le site de présentation du projet ainsi qu'une démo interactive qui simule l'expérience utilisateur.

## La démo

La maquette suit le parcours de **Stan**, 28 ans, utilisateur Premium qui découvre l'app pour la première fois.

Elle couvre :
1. Questionnaire d'onboarding (objectif, niveau, infos physiques, santé)
2. Révélation du coach personnalisé selon le profil
3. Tableau de bord
4. Location de kit avec scan QR
5. Séance guidée par le coach IA (timer, conseils, fin de séance)
6. Réservation de salle avec carte, agenda et cours collectifs
7. Profil utilisateur complet
